# PyQuantLib Examples

Jupyter notebooks demonstrating PyQuantLib usage.

Users familiar with QuantLib-Python (SWIG) will find the API similar.

## Available Examples

| Notebook | Description |
|----------|-------------|
| [01_option_pricing.ipynb](01_option_pricing.ipynb) | European option pricing with Black-Scholes |
| [02_modified_kirk_engine.ipynb](02_modified_kirk_engine.ipynb) | Spread options with Modified Kirk approximation |

## Running the Examples

### Prerequisites

```bash
pip install pyquantlib[examples]
```

### Launch

```bash
jupyter notebook
```
