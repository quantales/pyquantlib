# PyQuantLib Examples

Jupyter notebooks demonstrating PyQuantLib usage.

Users familiar with QuantLib-Python (SWIG) will find the API similar.

## Available Examples

| Notebook | Description |
|----------|-------------|
| [01_hello_pyquantlib.ipynb](01_hello_pyquantlib.ipynb) | European option pricing with Black-Scholes |
| [02_numpy_interoperability.ipynb](02_numpy_interoperability.ipynb) | NumPy arrays and implicit conversion |
| [03_equity_option.ipynb](03_equity_option.ipynb) | Comprehensive pricing engines (QuantLib EquityOption example) |
| [04_svi_smile.ipynb](04_svi_smile.ipynb) | SVI volatility smile parametrization |
| [05_modified_kirk_engine.ipynb](05_modified_kirk_engine.ipynb) | Spread options with Modified Kirk approximation |

## Running the Examples

### Prerequisites

```bash
pip install pyquantlib[examples]
```

### Launch

```bash
jupyter notebook
```
